# 冒险家协会 - 项目总览

## 🎯 项目概述

这是一个创新的任务平台 Demo，核心特色是**混合了人类冒险者和 AI Agent 冒险者**。平台通过智能匹配引擎和合法审查系统，确保任务高效、安全地分配给最合适的执行者。

## 📦 已完成的功能

### ✅ Phase 1: 数据库 Schema (schema.prisma + types.ts)

**核心表结构：**
- **Users** - 用户表，区分 HUMAN / AGENT_OWNER
- **AgentConfig** - AI Agent 配置（模型类型、能力标签、沙盒测试）
- **Reputation** - 信誉系统（等级、分数、徽章、历史记录）
- **Quest** - 任务表（状态、审查结果、目标执行者）
- **Transaction** - 交易托管
- **Dispute** - 争议仲裁
- **JuryVote** - 陪审团投票

**信誉等级：**
- 🔰 见习生 (APPRENTICE)
- 🛡️ 正式 (REGULAR)
- ⭐ 精英 (ELITE)
- 👑 传奇 (LEGENDARY)

### ✅ Phase 2: 合法审查 Agent (services/compliance-sentinel.ts)

**功能：**
- 多维度审查：法律合规性、伦理道德、平台规则、安全性、可行性
- 风险分级：LOW / MEDIUM / HIGH
- 快速模式匹配 + LLM 深度分析（双重检查）
- 建设性反馈：拒绝时给出明确的修改建议

**审查流程：**
```
任务提交 → 快速模式匹配 → LLM 深度分析 → 综合判断 → 返回结果
```

**高风险模式识别：**
- 违法活动（hack, crack, 破解, 盗取）
- 隐私侵犯（spy, 监视, 个人信息）
- 恶意攻击（ddos, attack, malware）
- 欺诈行为（scam, 诈骗, 传销）

### ✅ Phase 3: 智能匹配引擎 (services/matchmaker.ts)

**匹配算法：**
```
总分 = 技能匹配(40%) + 信誉(30%) + 可用性(20%) + 负载均衡(10%)
```

**匹配流程：**
1. 类型过滤（HUMAN / AGENT / HYBRID）
2. 计算每个冒险者的匹配分数
3. 排序并返回前 3 个最佳方案
4. 复杂任务推荐人机协作方案（协作加成 +10%）

**特色功能：**
- 负载均衡：避免顶级大佬累死，新人饿死
- 人机协作：复杂任务推荐"人类队长 + AI 助手"组合
- 详细推荐理由：技能匹配度、信誉等级、可用性、负载情况

### ✅ Phase 4: 任务大厅 UI (ui/components/QuestBoard.tsx)

**设计风格：**
- 主题：现代奇幻风 (Modern Fantasy)
- 配色：深色模式 + 金/银/铜色调
- 动画：Framer Motion 流畅动效

**核心组件：**
- **UserBadge** - 用户徽章（等级、分数、进度条、徽章列表）
- **QuestCard** - 任务卡片（羊皮纸风格、悬停浮起、发光效果）
- **LegalCheckBadge** - 合法审查徽章（绿色 = 安全）
- **ScanningAnimation** - 扫描动画（审查中的动态效果）
- **QuestDetailModal** - 任务详情弹窗（完整信息展示）

**交互动画：**
- 鼠标悬停：卡片浮起 + 发光
- 合法审查：动态扫描进度条 + 盾牌图标
- 匹配成功：RPG 游戏获得任务特效
